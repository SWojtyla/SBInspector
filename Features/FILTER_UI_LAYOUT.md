## Key UI Improvements

1. **Dynamic Layout**: The "Attribute Name" field only appears for Application Property filters
2. **Contextual Operators**: Only relevant operators are shown based on field type
3. **Appropriate Input Types**: 
   - Text input for strings
   - Number input for counts/sequences
   - Datetime-local picker for times
4. **Responsive Design**: Layout adapts to screen size using Bootstrap grid
5. **Visual Feedback**: Real-time message count updates as you type
6. **Clear Hierarchy**: Each filter is in its own card for clarity
7. **Easy Management**: Add/remove filters with clear buttons

## Behavior Notes

- All filters use AND logic (message must match ALL filters)
- Empty filters are ignored
